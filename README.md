# 💰 Finance Manager

A web-based Finance Management System built using Java, JSP, Servlets, and MySQL.

---
 
## 🚀 Features

- 👤 Customer Management (Add, View, Search, Delete)
- 💸 Loan Management
- 📊 EMI Tracking System
- ⚠️ Overdue Calculation
- 💰 Daily Collection Tracking
- 📧 Email Reminder System
- 📈 Dashboard Analytics

---

## 🛠️ Tech Stack

- Java (Servlets, JSP)
- MySQL Database
- JDBC
- Bootstrap (UI)
- HTML, CSS, JavaScript

---

## 📂 Project Structure
src/
└── main/
├── java/
│ ├── dao/
│ ├── model/
│ ├── servlet/
│ └── util/
├── webapp/
│ ├── components/
│ └── *.jsp
└── db.properties

---

## ⚙️ Setup Instructions

1. Clone the repository  
2. Import into Eclipse (Dynamic Web Project)  
3. Configure MySQL database  
4. Update `db.properties`  
5. Run on Apache Tomcat
